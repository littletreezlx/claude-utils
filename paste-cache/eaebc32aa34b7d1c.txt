# Role: Senior Flutter Frontend Engineer
# Task: Refactor the Chat Interface to match "Flame Design Language"

## Context
We are upgrading the UI of "LittleTree AI" from a basic Material 3 look to a custom "Soft Tech" aesthetic. The goal is to make the app feel warmer, more organic, and polished.

## Tech Stack Constraints
- Flutter 3.35+
- Packages: `flutter_animate`, `flutter_screenutil`, `riverpod`, `google_fonts`.
- Icons: Use `Lucide Icons` (via `lucide_icons` package) or `CupertinoIcons` for a thinner, more modern look if available, otherwise standard Material Symbols outlined.

## Design Tokens (Flame Language)
- **Primary Glow**: `Color(0xFFE65100)`
- **Warm Background**: `Color(0xFFFAFAF8)`
- **Soft Surface**: `Colors.white`
- **Text Main**: `Color(0xFF2C2C2C)`
- **Text Secondary**: `Color(0xFF757575)`

Use a `Row` layout:
- **Left**: `NavigationRail` (Custom implementation, not standard widget). Width: 260px.

- **Right**: Main Chat Area.

    - Background: `Primary.withOpacity(0.12)`.

**B. Chat Area & Bubbles**
- **User Bubble**:
    - Gradient Background: `LinearGradient(colors: [Color(0xFFFFA000), Color(0xFFE65100)], begin: Alignment.topLeft, end: Alignment.bottomRight)`.

- **Interaction**: When the user focuses the `TextField`, animate the Shadow to be stronger (`opacity: 0.2`) and `blurRadius: 30` to simulate "active energy".
