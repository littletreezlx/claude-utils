## 核心哲学: Warm Essentialism（FlameTree 默认母语示例,非全局规范）

1. **Less but Better**: 不做堆砌，只做精品。敢于砍掉非核心功能
2. **Warmth > Tech**: 技术隐形。追求"暖陶 (Warm Ceramic)"、"晨光"、"原木"质感
3. **Tactile Object**: 软件像精密机械旋钮或高级打火机——精准且克制的触觉愉悦
4. **视觉降噪**: 重要性 1/2 的元素，视觉比重降为 1/4（但守住 WCAG 对比度底线）

## 五维审美评审标准

### 1. Typography（排版与字体）

**核心问题**：字体有性格吗？

- Display Font 与 Body Font 是否形成对比？
- 字阶层级是否清晰（≥3 级区分）？
- 中文字体是否经过选择（不是系统默认了事）？
- 字重变化是否有节奏感（不是全部 Regular 或全部 Bold）？

**评级标准**：
- ★★★★★ 字体选择有明确性格，层级清晰，中英文搭配和谐
- ★★★☆☆ 系统字体但层级运用合理
- ★☆☆☆☆ 单一字重/字阶，无层级区分

### 2. Color & Theme（色彩与主题）

**核心问题**：忠于暖陶瓷基调的同时，有独特色彩个性吗？

- 主色是否强势、点缀色是否锐利？（拒绝均匀分布的"安全"配色）
- 阴影是否带环境色（暖色阴影 vs 纯黑/纯灰）？
- 背景是否有温度（接近 #FFF8F0 暖底 vs 生硬纯白 #FFFFFF）？
- 色彩对比度是否兼顾可读性和氛围感？

**评级标准**：
- ★★★★★ 暖陶瓷基调 + 独特个性 + 环境色阴影 + 点缀色精准
- ★★★☆☆ 基调正确但配色安全无趣
- ★☆☆☆☆ 冷灰调 / 纯黑阴影 / 无环境色

### 3. Spatial Composition（空间构成）

**核心问题**：敢于不对称吗？留白是设计语言还是"没设计"？

- 是否有打破网格的元素（overlap、对角线流动）？
- 留白是否有意图（烘托核心 vs 单纯空旷）？
- 信息密度是否适当（不拥挤也不空洞）？
- 核心组件是否有"C 位"感？

**评级标准**：
- ★★★★★ 构图有张力，留白有意图，核心元素 C 位感强
- ★★★☆☆ 规整但缺乏个性
- ★☆☆☆☆ 均匀间距 + 对称居中 + 无层次

### 4. Material & Texture（材质与质感）

**核心问题**：有氛围深度吗？

- 是否有 Subsurface Glow（透射深度）而非单薄的纯色？
- 是否运用了 gradient mesh / noise texture / layered transparency？
- 阴影是否多层且柔和（焦糖色弥散 vs 单层硬阴影）？
- 背景是否有氛围处理（不是空白画布）？
- 微光边框、毛玻璃等是否增强了质感而非装饰？

**评级标准**：
- ★★★★★ 陶瓷般温润深度 + 多层质感 + 环境阴影到位
- ★★★☆☆ 有基本质感但缺乏深度
- ★☆☆☆☆ 纯色卡片 + 单层阴影 + 无材质感

### 5. Motion & Haptics（动效与触觉）

> 注意：静态截图无法直接评判动效，此维度基于视觉线索推断 + 标注 `[需人工体验]`。

**核心问题**：关键时刻有精心编排吗？

- 页面加载 / 状态切换是否暗示了 stagger reveal 的可能？
- UI 元素是否有状态区分（idle / hover / pressed / disabled）？
- 是否预留了弹簧物理曲线的空间（元素间有层次感）？
- 触觉反馈点是否合理（关键操作 vs 过度反馈）？

**评级标准**：
- ★★★★★ 状态区分清晰 + 视觉暗示动效编排 + 触觉反馈设计合理
- ★★★☆☆ 有基本状态区分
- ★☆☆☆☆ 无状态变化 / 纯静态感

## 反 AI 味清单

以下模式**不是绝对禁止**，但使用时必须有明确的"为什么"。如果找不到理由，大概率是 AI 默认审美在作祟。

### 配色反模式
- 紫色渐变 + 白底（AI 生成的经典配色）
- 蓝紫渐变 CTA 按钮（每个 AI demo 的默认）
- 灰色系"安全"配色（不出错也不出彩）

### 布局反模式
- 圆角卡片网格 + 浅灰背景（最常见的 AI layout）
- 所有元素均匀间距、对称居中（安全但无趣）
- Hero Section + 三列 Feature Cards（着陆页模板）

### 字体反模式
- Inter / Roboto / Arial / system-ui 作为装饰字体（无性格）
- 全局统一字重（无节奏感）

### 动效反模式
- 纯装饰性动画（没有信息传达或状态反馈意义）
- 所有元素同时 fade-in（无编排层次）

### 质感反模式
- 纯白卡片 + 单层灰阴影（无材质深度）
- 生硬的纯色背景（无氛围处理）

## Signature Moment 概念

每个页面至少应有一个让用户**记住**的瞬间——不是花哨的动画，而是"这个细节让我想再用一次"。

**好的 Signature Moment 特征**：
- 服务于核心功能（不是装饰）
- 精准且克制（不是炫技）
- 独一无二（不是通用组件库能提供的）

**例子**：
- FlameTree Pick 的抽选动画：三段阻尼物理 + 精准触觉 = 机械旋钮感
- 温度感的微交互：卡片选中时的呼吸光效 = 陶瓷釉面反光

**评审时**：如果一个页面找不到 Signature Moment，这本身就是一个重要的设计缺失。
