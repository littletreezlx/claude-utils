## 规则

写 POS 竞品对比文档(以及其他给产品负责人读的文档)时,**严格限制 emoji 使用**:

- **禁止**:章节标题、定位标签、行文中的装饰 emoji,如 🦾 全能型 / 🎯 操控入口 / 📋 管控型 / 🏆 / 🚀 / 💡 / 🔥 / ✨ 等
- **勉强可接受**:表格格子内用作**状态符号**的 ✅ ❌ ❓ ⚠️(做能力矩阵、支持情况的可视化标记)
- **原则**:emoji 只在作为信息载体时可以出现,用作视觉装饰或强调时必须删除

## Why

2026-04-17 远程协助对比文档中用了 🦾 全能型 / 🎯 操控入口 / 📋 管控型 做定位标签,被 Founder 纠正"文档不要太多 emoji"。装饰性 emoji 会让文档显得不正式、不够专业,且对 AI 跨会话阅读无信息增益,纯粹是视觉噪音。这条审美偏好适用于整个项目的对比文档。

## How to apply

- 新建或修改 `docs/*-对比.md`、`*/docs/*.md` 任何正文段落时,检查是否出现装饰性 emoji —— 有则去掉,用纯文字标签(如"全能型""操控入口""管控型")替代
- 写"一句话总结""核心洞察"等小节时,不要用 emoji 做 bullet 前缀,直接用加粗或破折号
- 表格中的 ✅/❌/❓ 保留,但不要在表格外的正文里重复使用这些符号做强调
- 如果不确定某个 emoji 是"信息"还是"装饰",判断标准:去掉它信息是否丢失?不丢失就是装饰,必须删
