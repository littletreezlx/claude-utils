## 文档组织规范

**横向对比文档** → `项目根目录/docs/[功能模块]/README.md`
- 多家竞品同一功能的技术对比
- 技术路线、实现差异、功能定位总结
- 为产品决策和自研参考提供输入

**各家实现细节** → 保留在各自项目目录 `*/src/`
- 不复制源码
- 文档记录代码路径索引

**临时分析文件** → `~/_scratch/`
- 分析过程中的调试日志、原始结论
- 整理后提炼写入对比文档，原文件可删除
