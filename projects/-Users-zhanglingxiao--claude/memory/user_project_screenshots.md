用户几乎所有项目都将截图（包括 AI 生成工具的设计产出、App 实际截图等）放在项目根目录的 `screenshots/` 目录下，且会持续更新。

示例路径：`~/LittleTree_Projects/flutter/flametree_rss_reader/screenshots/`

执行 UI 相关工作（ui-vs、ui-vision-check、ui-vision-advance 等）时，直接去 `screenshots/` 查找，不要询问用户截图位置或如何对比。
