用户除了资深全栈工程师身份外，也是活跃的 A 股/港股投资者。

**持仓风格**: 哑铃型配置
- 防守端：紫金矿业、铜陵有色、海螺水泥（通胀对冲/高股息）
- 进攻端：人工智能 ETF、恒生互联网、立讯精密（科技/AI/出海）
- 弹性仓：中矿资源（锂矿周期）

**核心痛点**:
- 知行不一：中长线逻辑选股，但受单日涨跌刺激做情绪化操作（如恐慌中清仓 ETF 追个股）
- 信息盲区：春节脱网期间错过伊朗战争预警，盈利全部吐回
- 方法论错配：曾用短线震荡指标（KDJ）指导趋势型价值股

**How to apply**: 投资模块的设计和实现应时刻围绕这些痛点。用户需要的是"强制遵守自己规则的系统"，而非更多信息轰炸。
