## 角色

独立 Flutter 开发者，使用 Claude Code 作为主要开发工具。管理 7 个 Flutter 应用 + 1 个共享框架 (flutter_common)。

## 技术偏好

- **务实 > 完美**：可维护性优先，不追求完美架构
- **破坏性方案优先**：倾向于彻底重做而非修补，除非用户明确反对
- **AI 友好设计**：脚本输出、文档结构都为 AI 可解析优化
- **中文沟通**：使用中文回答，代码注释/日志保留英文

## 工作方式

- 品牌名: FlameTree（火焰树）系列
- 项目管理: 无外部工具（Jira/Linear），全在 TODO.md + 会话中管理
- 发布: 本地 flutter build，不用 CI/CD
- 协作: 独立开发，偶尔用 Gemini API 做产品/设计咨询

## 决策风格

- 给出方案时倾向于"先做再说"而非过度讨论
- 重视投入产出比，会按 P0-P3 分优先级
- 喜欢系统化思维（工具链、自动化脚本、文档体系）
